# RELIFE PROJECT

## Requirements
* For development, you only need Node.js and yarn installed in your environment.

## Install

 	 $ git clone https://github.com/Atlantis-Fellowship-Instituto-Atlantico/Relife-front.git

 	 $ cd Relife-front


## Start & watch

  $ yarn start

## Languages & tools
* React
* React-router-dom
* React-input-mask
* React-hook-form
* yup
* Typescript
* CSS
* Material-ui
